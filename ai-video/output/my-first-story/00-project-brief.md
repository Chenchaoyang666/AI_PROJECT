# 雨夜里 一个女孩站在城市

## One-line Goal

Create a 18-second AI cinematic short for Douyin / Reels / Shorts, using 9:16 framing and a single main character in one primary location.

## Logline

雨夜里，一个女孩站在城市街头，最后转身离开。

## Character Lock

- Name: Lin
- Appearance: young woman, sharp bob haircut, pale skin, restrained expression
- Wardrobe: dark long coat, understated boots, minimal accessories
- Mood baseline: quiet, observant, emotionally contained
- Consistency rule: Keep the same face shape, hairstyle, outfit silhouette, and key lighting ratio across all shots.

## Visual Direction

- Primary style: cinematic realism, moody blue-red lighting, soft contrast
- Color palette: blue neon, red accents, silver reflections, wet asphalt glow
- Environment rule: Keep the same rainy city street, same neon palette, and restrained background movement throughout.
- Motion rule: Each shot should focus on one simple action with natural motion and no exaggerated gestures.

## Production Defaults

- Aspect ratio: 9:16
- Target duration: 18s
- Dialogue/subtitles language: zh-CN
- Music direction: slow atmospheric electronic with distant city ambience
- Export target: 1080x1920, H.264, clean master first, then subtitle version.

## Execution Path

1. Generate one key image per shot.
2. Turn each key image into a 5-10 second AI video clip.
3. Keep only the cleanest take for each shot.
4. Assemble the final cut in CapCut with simple cuts or short fades.
5. Add BGM, ambience, subtitles, and a light grade at the end.
